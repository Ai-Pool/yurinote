Yurinote 隐私权政策 (Privacy Policy)

最后更新日期：2026年4月 (Last Updated: April 2026)

欢迎使用 Yurinote（以下简称“本应用”）。我们重视您的隐私，并致力于保护您的个人数据。
Welcome to Yurinote ("the App"). We value your privacy and are committed to protecting your personal data.

1. 我们访问的信息（Information We Access）

  • 位置信息：用于获取天气信息，仅在用户授权后使用，不会存储。
    Location Data: Used to provide weather features. Accessed only with user permission and not stored.
  • 本地存储：用于在用户设备上保存日记内容与附件。
    Local Storage: Used to store journal content and attachments on the user's device.
  • Google 账户数据：当用户启用云端备份或日历同步时，本应用会访问用户的 Google Drive 和 Google Calendar，仅限于本应用创建的数据。
    Google Account Data: When enabled, the App accesses Google Drive and Google Calendar to store and sync app-created data only.

2. 数据的使用方式（How We Use Data）

我们仅在提供应用核心功能时使用数据：
We only use data to provide core application features:

  • 将日记备份至用户的 Google Drive / Backup data to Google Drive
  • 将日记同步至用户的 Google Calendar / Sync data to Google Calendar

我们不会将用户数据用于广告、分析或任何与应用功能无关的用途。
We do not use user data for advertising, analytics, or any unrelated purposes.

3. 数据存储（Data Storage）

本应用不会在开发者服务器上存储任何用户数据。
We do not store any user data on developer servers.

所有数据存储在以下位置：
All data is stored in:

  • 用户设备本地 / User's local device
  • 用户自己的 Google 账户（Google Drive / Google Calendar） / User’s own Google account (Google Drive / Google Calendar)

我们无法访问或控制用户在其 Google 账户中存储的数据内容。
We do not control or access data stored in the user’s Google account beyond authorized use.

4. 数据保护（Data Protection）

我们不会在服务器端存储用户数据。所有通过 Google API 访问的数据仅用于实时处理，不进行持久化存储。
We do not store data on servers. Data accessed via Google APIs is processed in real time and not retained.

我们采用行业标准的安全措施，包括：
We implement industry-standard security measures, including:

  • HTTPS 加密传输 / HTTPS encryption
  • 遵循 Google API 安全最佳实践 / Following Google API security best practices

5. 数据保留与删除（Data Retention & Deletion）

我们不会保留任何 Google 用户数据。
We do not retain any Google user data.

所有数据均由用户控制，并存储在其设备或 Google 账户中。
All data is controlled by the user and stored on their device or Google account.

用户可以通过以下方式停止数据访问：
Users can stop data access by:

  • 在 Google 账户中撤销本应用权限 / Revoking access via their Google account permissions

一旦撤销授权，本应用将无法再访问用户数据。
Once access is revoked, the App can no longer access user data.

如果用户对数据处理或删除有任何疑问，可通过下方联系方式与我们联系。
If users have questions about data handling or deletion, they may contact us below.

6. 第三方服务（Third-Party Services）

本应用使用 Google AdMob 提供广告服务，该服务可能使用广告标识符。
The App uses Google AdMob, which may use advertising identifiers.

7. Google API 使用合规声明（Limited Use Disclosure）

本应用遵守 Google API Services User Data Policy，包括 Limited Use 要求。
The App complies with the Google API Services User Data Policy, including Limited Use requirements.
